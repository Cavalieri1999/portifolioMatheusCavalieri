HTML5 (HyperText Markup Language version 5) é a versão mais recente da linguagem de marcação usada para estruturar e apresentar conteúdo na web. Ele introduziu novas funcionalidades, como suporte nativo a multimídia (vídeo e áudio), elementos semânticos (como article, section, e header), e maior interatividade por meio da integração com APIs do navegador, como o Canvas para gráficos. Além disso, o HTML5 visa melhorar a acessibilidade, otimizar o carregamento de páginas e eliminar a necessidade de plugins externos, como o Flash. CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência e o layout de páginas web. Ele permite estilizar elementos HTML, controlando aspectos como cores, fontes, margens, espaçamentos, tamanhos, e posicionamentos. Com o CSS, é possível criar layouts responsivos, ajustar a exibição para diferentes dispositivos e resoluções, além de aplicar animações e transições. O CSS separa o conteúdo (HTML) da apresentação, facilitando a manutenção e a reutilização de estilos em várias páginas de um site. JavaScript (JS) é uma linguagem de programação usada para adicionar interatividade e dinamismo a páginas web. Com ele, é possível manipular elementos HTML e CSS em tempo real, responder a eventos do usuário (como cliques e teclas), validar formulários, criar animações, e carregar dados de servidores sem recarregar a página (usando AJAX). JavaScript é amplamente usado em conjunto com HTML e CSS para criar experiências de usuário mais ricas e interativas na web. UX (User Experience) e UI (User Interface) são conceitos centrais no design de produtos digitais. UX (Experiência do Usuário) se refere à qualidade da interação do usuário com um produto, serviço ou sistema. Envolve todo o processo de design para garantir que o produto seja intuitivo, eficiente e satisfatório de usar. O foco está na usabilidade, acessibilidade e na satisfação do usuário ao navegar pelo sistema. UI (Interface do Usuário) é a parte visual e interativa do design, ou seja, a interface que o usuário vê e com a qual ele interage. Inclui elementos como botões, menus, ícones, cores e tipografia, garantindo que o layout seja esteticamente agradável e funcional. Enquanto o UX foca na experiência e funcionalidade, o UI está relacionado ao visual e à interação direta com o produto. C# (pronúncia-se "C Sharp") é uma linguagem de programação moderna, orientada a objetos e fortemente tipada, desenvolvida pela Microsoft em 2000 como parte de sua plataforma .NET. Ela foi projetada para ser simples, eficiente e robusta, combinando a simplicidade do C e C++ com o poder da programação voltada para o objeto Um dos principais objetivos do C# é fornecer uma linguagem que seja fácil de aprender e usar, mas que também permita a criação de aplicações complexas, como programas de desktop, jogos, aplicativos móveis e web, e sistemas empresariais. A linguagem é amplamente utilizada no desenvolvimento de aplicativos Windows e na criação de aplicativos web dinâmicos usando o framework ASP.NET. SQL (Structured Query Language) é uma linguagem de programação especialmente projetada para gerenciar e manipular bancos de dados relacionais. Foi criado nos anos 1970 pela IBM e tornou-se o padrão internacional para interação com bancos de dados, sendo amplamente utilizado até hoje em diversos sistemas de gestão de dados, como MySQL, PostgreSQL, SQL Server, Oracle Com SQL, desenvolvedores e analistas de dados podem extrair e manipular grandes volumes de dados de forma eficiente, sendo uma habilidade essencial para trabalhar com bancos de dados em qualquer aplicação que envolva armazenamento e recuperação de informações.
